My Website Name:
      My website name is AnimeHub.

Anime Hub:
       It is a modern anime streaming website built with a dark, immersive UI inspired by popular anime platforms.
       It focuses on clean design, smooth interactions, and an eye-catching color theme that fits the anime vibe perfectly.

About the Project:
          Anime Hub is a front-end anime streaming website where users can browse anime content in a visually appealing layout. 
          The design uses a dark theme with accent colors to highlight important elements like buttons, links, and cards.
          This project is built to practice UI design, layout structure, and front-end development for a real-world streaming-style website.

Color Theme:
        The website uses a dark theme with orange accents:
                    Background: #1b1b1b
                    Secondary Background (Header / Footer / Cards): #292929
                    Primary Accent: #ffa31a
                    Text Color: #808080
                    Headings: #ffffff

        This combination gives the site a bold, cinematic anime look.

Features:
        Dark anime-inspired UI
        Responsive header and footer
        Anime cards for episodes/videos
        Smooth hover effects on links and buttons
        Clean and readable layout
        Scalable design for future features

Tech Stack:
        HTML5
        CSS3
        JavaScript
        Tailwind CSS
